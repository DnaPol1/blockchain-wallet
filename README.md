# Blockchain Wallet (Educational Project)
Учебный проект для понимания базовых принципов работы блокчейна: создание транзакций, майнинг блоков, генерация ключей и персистентность данных.
## Реализованный функционал:
1. Генерация криптографических ключей и адресов кошелька.
2. Создание и подписывание транзакций.
3. Механизм майнинга блока с RoW.
4. Сохранение состояния блокчейна в JSON-файлы.
5. Простой веб-интерфейс для взаимодействия.
## Запуск
`pip install -r requirements.txt`

`uvicorn app:app --reload`
